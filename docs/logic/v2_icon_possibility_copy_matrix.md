# v2 Icon 可能性与文案矩阵

日期：2026-05-30

目的：把 32 个 icon 从“图标资源”升级为“命理结果可映射的人生可能性”。前端展示给用户的是处境、选择、取舍和下一步；命理字段只作为内部解释依据。

重要边界：

- Icon 只代表“信号类型”，不代表固定命运结论。
- 节点只代表地图槽位，不代表固定人生意义。
- 每个 icon 出现在什么节点、解释成什么处境，必须由用户的生辰八字、节气、干支、五行气、规则分数和选择路径计算。
- LLM 只负责把已计算结果改写成人话，不能自行决定 icon、路线或终局。
- 下一版视觉重绘标准见 `docs/logic/v2_map_icon_redraw_standard.md`。

## 1. 语言原则

1. 不说定命，改说趋势和火候。
2. 不说系统字段，改说用户能感知的处境。
3. 不说“已选路线”，改说“你现在面临的选择”。
4. 路线名用白话，解释层再放命理语言。
5. 每个 icon 都必须回答四件事：你会遇到什么、这代表什么、适合怎么做、不宜怎么做。
6. 每个 icon 的解释必须绑定当前节点和用户数据，不能复用成固定模板。

## 2. 页面总话术

### 2.1 当下总览

标题：

```text
你现在面临的选择
```

短句：

```text
你站在起点，前路有几种走法。先看清当下处境，再决定哪条路更适合你。
```

主建议标签：

```text
更适合先走
```

其他选择标签：

```text
另一种可能
```

点击某条路线后：

```text
你正在查看
```

### 2.2 Icon 详情模板

```text
你会遇到：{人生处境}
它提醒你：{命理信号翻译成人话}
适合：{一件可执行的小动作}
不宜：{需要避开的动作}
这条路的意思：{取舍总结}
```

### 2.3 路线白话命名

| 内部路线 | 用户看到 | 解释层可用 |
|---|---|---|
| A | 稳住主线 | 守主脉、气不散、先稳局 |
| B | 试新机会 | 开一扇门、顺风、火候初起 |
| C | 放慢绕行 | 暗流、避冲、留余地 |
| ROOT | 补齐条件 | 补根基、打底盘、先蓄气 |

## 3. 32 个 Icon 可能性

### 01. `momentum_boost`

- 用户看到：推进机会
- 内部含义：推进力高，行动能量上升。
- 人生可能性：一件事开始有推力，适合把停住的事情往前挪一格。
- 命理触发：momentum 高；今日五行同频或生助；宜项偏行动、出行、交易、会友。
- 适合：先推进一个可验证的小动作。
- 不宜：一次把承诺、预算、关系全部推满。
- 短文案：现在有推力，可以往前一格。
- 展开文案：这不是让你全速冲，而是说明气势已经起来。先选一个最容易收到反馈的动作，把局面推开一点，再看后面的路是否变亮。

### 02. `stability_anchor`

- 用户看到：稳住基础
- 内部含义：稳定性、容错率、底盘承重。
- 人生可能性：当前最重要的不是变化，而是把基础条件压稳。
- 命理触发：stability 高时适合守成；stability 低时提示先补底盘。
- 适合：整理资源、时间表、现金流和手上承诺。
- 不宜：在基础未稳时开新局。
- 短文案：先把脚下站稳。
- 展开文案：这一格提醒你先看底盘。很多选择看似是方向问题，实际是资源、作息、承诺和节奏没有放平。基础稳住，后面的分枝才有余地。

### 03. `support_lighthouse`

- 用户看到：有人照路
- 内部含义：外部支持、贵人、资源入口。
- 人生可能性：你不是只能独自推进，旁边有人、信息或资源可以借力。
- 命理触发：support 高；今日五行生我；宜会友、纳采、订盟、求助、沟通。
- 适合：找关键人确认、请教、递出一个请求。
- 不宜：把支持理解成别人会替你承担后果。
- 短文案：有灯可借，先问清楚。
- 展开文案：这盏灯代表外部助力。它不保证事情一定成，但说明你可以借人、借信息、借场域。动作要短，问题要准，先拿到一个明确回应。

### 04. `relationship_bridge`

- 用户看到：关系可通
- 内部含义：关系顺畅、沟通桥梁。
- 人生可能性：某段关系有机会重新接上，或通过一次沟通降低误会。
- 命理触发：relationship 高；support 不低；风险未压过关系分。
- 适合：释放善意、说清一个重点、建立共同语境。
- 不宜：翻旧账、连续追问、要求对方立刻表态。
- 短文案：桥还在，话可以轻一点说。
- 展开文案：这座桥说明关系没有完全断。适合先让气流动起来，而不是马上定输赢。说一个重点，留一点空间，比一次说尽更容易过桥。

### 05. `wealth_safe_chest`

- 用户看到：守住资源
- 内部含义：财富安全、资源承诺安全度。
- 人生可能性：钱、资源、时间或信用需要先守住边界。
- 命理触发：wealthSafety 低或 risk 高；财富问题权重上升。
- 适合：小额、可撤、先验证。
- 不宜：大额承诺、情绪消费、不可逆交易。
- 短文案：财箱先扣紧。
- 展开文案：这一格不是说不能动钱，而是提醒你先看安全边界。钱、时间和承诺都是资源，今天更适合小步验证，不适合一次押满。

### 06. `risk_barrier`

- 用户看到：前方有阻
- 内部含义：风险干扰、冲突、误判、情绪波动。
- 人生可能性：路面有阻力，强推会让后面更难收。
- 命理触发：risk 高；今日冲煞；忌项命中争执、破土、重承诺。
- 适合：降速、复核信息、把决定拆小。
- 不宜：硬闯、争输赢、临时大改方向。
- 短文案：前方有阻，先别硬推。
- 展开文案：路障代表阻力已经显形。它不等于没路，而是提醒你这一段要少做不可逆动作。先把事实核清楚，再决定是否继续。

### 07. `career_path`

- 用户看到：事业上坡
- 内部含义：事业推进、项目、工作、面试、汇报、合作。
- 人生可能性：你正走到一段需要交付成果的坡。
- 命理触发：intent=career；momentum/support 对事业权重上调。
- 适合：交付一个可见成果，争取一次明确反馈。
- 不宜：只规划不落地，或同时开太多线。
- 短文案：事业坡要靠成果上去。
- 展开文案：这一坡看的是可交付。你不需要马上证明全部能力，只需要把最能代表自己的一个小结果做出来，让别人看见你正在往上走。

### 08. `money_commitment`

- 用户看到：钱与承诺
- 内部含义：财富承诺、交易、消费、买卖、合约。
- 人生可能性：你面前有一笔资源调度，核心是承诺边界。
- 命理触发：intent=wealth；wealthSafety 与 risk 共同决定。
- 适合：先算要承受什么、确认可撤条件、分批推进。
- 不宜：口头答应后再补规则。
- 短文案：先看承诺，再看回报。
- 展开文案：这不是单纯的钱的问题，而是钱背后的承诺。先问清楚谁承担、能不能撤、何时交付，再决定要不要往前走。

### 09. `relationship_contact`

- 用户看到：先说清楚
- 内部含义：关系沟通、联系、复合、互动窗口。
- 人生可能性：关系议题正在要求一次更清楚的表达。
- 命理触发：intent=relationship；relationship/support/risk 联合判断。
- 适合：用短句表达真实需求。
- 不宜：试探、冷处理、把情绪包装成问题。
- 短文案：先把一句话说清楚。
- 展开文案：这封信代表沟通窗口。适合说真实、简单、可回应的话。不需要一次聊完关系，只要先让对方知道你真正想确认什么。

### 10. `mood_rest`

- 用户看到：回到状态
- 内部含义：情绪支持、休整、补能。
- 人生可能性：当前的重点不是做更多，而是先把自己从消耗里拿出来。
- 命理触发：intent=mood；momentum 降、stability 权重升、risk 降。
- 适合：减少输入、睡眠、整理空间、完成一件小事。
- 不宜：在情绪乱时做长期决定。
- 短文案：先回气，再看路。
- 展开文案：这朵云提醒你先回到状态。很多时候不是路不好，而是人太耗。先让身体和情绪稳下来，判断才会恢复。

### 11. `direction_choice`

- 用户看到：到了岔口
- 内部含义：方向选择、变道、离职、转向、改变。
- 人生可能性：你正在面对两个或多个方向，不能只靠冲动选。
- 命理触发：intent=decision；laneChange 状态进入判断。
- 适合：列出可逆动作和不可逆动作，先做可逆测试。
- 不宜：把犹豫等同于失败。
- 短文案：先看哪条路能试错。
- 展开文案：岔口不是让你立刻定终局，而是让你分清哪条路能先试、哪条路一走就很难回头。先选可验证的一步。

### 12. `today_general`

- 用户看到：看全局
- 内部含义：今日综合、默认意图。
- 人生可能性：没有单一问题时，先看整体天气和路面。
- 命理触发：intent=general；无明确事业/财富/关系/情绪关键词。
- 适合：安排轻重缓急，先做最顺的一件事。
- 不宜：把所有问题都压到今天解决。
- 短文案：先看今天整体风向。
- 展开文案：罗盘代表全局判断。它不替你决定人生，只告诉你今天哪边更顺、哪边更费力。先顺势做一件事，别把一天塞满。

### 13. `opportunity_window`

- 用户看到：窗口打开
- 内部含义：机会窗口、短期入口。
- 人生可能性：某个机会出现，但窗口不一定持续很久。
- 命理触发：momentum/support 高且 risk 可控；selectedRoute 倾向 B。
- 适合：小步试探、发出申请、递出作品、约一次沟通。
- 不宜：因为看见机会就立刻押全局。
- 短文案：门开了一线，先试一步。
- 展开文案：窗口打开代表时机有一点亮。它不是终点，而是入口。先做一个轻动作，让对方或环境给你反馈。

### 14. `red_risk_segment`

- 用户看到：红色路段
- 内部含义：风险路段、阻力集中段。
- 人生可能性：某段路可以走，但路面更窄，失误成本更高。
- 命理触发：risk 中高；冲煞或忌项命中；但未到 blocked_path。
- 适合：慢行、留证据、拆步骤。
- 不宜：带情绪推进、临时变道、口头承诺。
- 短文案：这段能走，但要慢。
- 展开文案：红色路段不是封路，而是提醒你这里容错偏低。把动作拆小，重要信息留痕，走过去再加速。

### 15. `slow_down`

- 用户看到：先慢下来
- 内部含义：降速通过。
- 人生可能性：你需要用慢一点的速度换判断质量。
- 命理触发：risk 偏高或 stability 偏低；speedAdvice=slow_down。
- 适合：暂停一拍、复核、先做低风险事项。
- 不宜：为了显得果断而强行推进。
- 短文案：慢一点，反而更稳。
- 展开文案：降速不是退缩，而是把局面看清。今天适合先处理确定性高的小事，等路面稳定再做大动作。

### 16. `keep_lane`

- 用户看到：沿主路走
- 内部含义：保持车道、继续当前主线。
- 人生可能性：当前方向没有明显错误，重点是稳定推进。
- 命理触发：stability 较好；laneChange=keep_lane；A 路更优。
- 适合：继续已有计划，做小幅优化。
- 不宜：因为焦虑而频繁换方向。
- 短文案：主路可走，先别乱变道。
- 展开文案：这条线说明你的主方向还可以承载。不要因为短期波动就重开人生议题，先把已有路线走扎实。

### 17. `lane_change`

- 用户看到：可以转向
- 内部含义：可以变道、条件允许调整路线。
- 人生可能性：有调整空间，但需要看清窗口和要承受的部分。
- 命理触发：momentum/support 高，risk 未过高；laneChange=can_change。
- 适合：试一个侧枝，保留回主路的接口。
- 不宜：把试探直接升级成决裂或彻底转身。
- 短文案：可以转向，但先留回路。
- 展开文案：变道不是换命，而是调整走法。适合小范围试新方向，同时保留原来的安全接口。

### 18. `detour_route`

- 用户看到：绕开消耗
- 内部含义：绕行、避开噪音和风险。
- 人生可能性：直路阻力偏高，绕一下能保住后面的选择。
- 命理触发：risk 高或 mood intent；C 路更优。
- 适合：避开争执、降低承诺、换一个入口。
- 不宜：把绕行理解成放弃。
- 短文案：绕一下，是为了后面更好走。
- 展开文案：绕行代表保留余地。当前不是不能前进，而是直推会消耗太多。换一条柔一点的路，后面还可以合流。

### 19. `accelerate_now`

- 用户看到：现在加一格
- 内部含义：现在小步加速。
- 人生可能性：行动窗口已经出现，可以比平时多推一点。
- 命理触发：momentum 高、risk 低、support 不弱；speedAdvice=accelerate_now。
- 适合：推进一个明确节点，争取一次反馈。
- 不宜：把小步加速变成全局冲刺。
- 短文案：现在可以加一格。
- 展开文案：风已经起来，但还不是让你失控。把力气集中在一个节点上，快一点拿到反馈，再决定是否继续。

### 20. `accelerate_later`

- 用户看到：等等再加
- 内部含义：稍后小步加速。
- 人生可能性：方向可能有机会，但现在还差一个信号。
- 命理触发：momentum 中等、risk 或信息不明；speedAdvice=accelerate_later。
- 适合：先等反馈、补材料、确认关键人。
- 不宜：因为焦虑提前冲。
- 短文案：再等一个信号。
- 展开文案：沙漏代表火候还差一点。你可以准备，但先别把动作推到不可逆。等关键反馈出现，再轻轻加速。

### 21. `wait_signal`

- 用户看到：等信号亮
- 内部含义：等待信号、信息不清。
- 人生可能性：当前缺少关键事实或对方态度，贸然判断容易偏。
- 命理触发：support 不明、risk 中高、tunnelState 未转清。
- 适合：问清一个关键问题，等待明确回应。
- 不宜：用猜测替代事实。
- 短文案：信号未亮，先别抢。
- 展开文案：红绿灯提醒你看信号。不是不能走，而是现在最缺的是确认。等一句明确回应，比现在硬判断更有用。

### 22. `near_exit`

- 用户看到：出口快到了
- 内部含义：出口将近、视野转清。
- 人生可能性：一段不清楚的路快要走完，答案开始浮出来。
- 命理触发：tunnelState=near_exit；support/stability 改善。
- 适合：收尾、复盘、准备下一步。
- 不宜：临近出口时突然大改路线。
- 短文案：快出隧道了，先稳住。
- 展开文案：出口将近代表视野正在打开。现在不需要突然换方向，先把前面这段走完，到了亮处再看新选择。

### 23. `blocked_path`

- 用户看到：这段先不走
- 内部含义：暂不推进、路径暂时阻断。
- 人生可能性：某个方向短期不适合继续推进。
- 命理触发：risk 极高、信息缺失严重、或安全边界触发。
- 适合：暂停、撤回、换更低风险路径。
- 不宜：为了证明自己而硬闯。
- 短文案：这段先不走。
- 展开文案：断桥代表当前路径承载不了继续推进。暂停不是失败，而是避免把问题走成更重的负担。先换到安全路面。

### 24. `safe_zone`

- 用户看到：回安全区
- 内部含义：安全区、休整、补能、保持状态。
- 人生可能性：你需要先回到一个稳定的位置，再判断下一步。
- 命理触发：mood intent、stability 低、risk 高、momentum 低。
- 适合：休息、整理、找可靠的人说清楚。
- 不宜：在耗竭状态下做长期承诺。
- 短文案：先回到安全区。
- 展开文案：安全区代表先把自己放稳。不是所有问题都要今天解决，先让状态恢复，路况会更容易看清。

### 25. `wood_growth`

- 用户看到：新芽在长
- 内部含义：木，生长、启动、扩展。
- 人生可能性：新想法、新关系或新计划正在萌芽。
- 命理触发：日主木、今日木旺、木为助力或当前需要生发。
- 适合：启动、学习、建立联系、种下长期线索。
- 不宜：刚发芽就要求结果。
- 短文案：新芽刚起，先养起来。
- 展开文案：木象代表生长。它说明这件事有生机，但还小。今天适合种下线索、建立连接，不适合急着收成。

### 26. `fire_energy`

- 用户看到：火候上来
- 内部含义：火，能量、表达、曝光、热度。
- 人生可能性：表达欲和行动热度上升，容易被看见。
- 命理触发：日主火、今日火旺、momentum 上升，同时需看 risk。
- 适合：表达、展示、争取关注、短促推进。
- 不宜：情绪上头、话说太满。
- 短文案：火候上来，话要准。
- 展开文案：火象代表能量和被看见。今天可以亮相，但要控火。表达越短、越准，越不容易烧过头。

### 27. `earth_base`

- 用户看到：把地基垫高
- 内部含义：土，地基、承载、稳定。
- 人生可能性：事情要落地，需要更厚的条件支撑。
- 命理触发：日主土、今日土旺、stability 权重上升或需要补基础。
- 适合：整理结构、补资料、确认规则。
- 不宜：基础不清就盖高楼。
- 短文案：先补地基，再往上走。
- 展开文案：土象代表承载。它提醒你先把规则、资源、时间和责任放稳。地基越清楚，后面的选择越不容易塌。

### 28. `metal_order`

- 用户看到：立好规则
- 内部含义：金，秩序、边界、判断、收束。
- 人生可能性：需要用规则和边界把局面收住。
- 命理触发：日主金、今日金旺、关系或财富问题需要边界。
- 适合：定标准、写清条款、划边界。
- 不宜：只靠感觉判断。
- 短文案：规则清楚，路才清楚。
- 展开文案：金象代表秩序。它不是冷，而是让事情有边界。今天适合把话、钱、责任和期限说清楚。

### 29. `water_flow`

- 用户看到：顺水而行
- 内部含义：水，流动、信息、适应、缓冲。
- 人生可能性：局面正在变化，顺势调整比硬控更有效。
- 命理触发：日主水、今日水旺、support 信息流动强。
- 适合：观察、沟通、换入口、跟随反馈。
- 不宜：用固定想法压住变化。
- 短文案：水在流，先顺势看。
- 展开文案：水象代表流动。它提醒你不要急着把事情钉死，先看信息怎么流、关系怎么动，再决定往哪里靠。

### 30. `season_node`

- 用户看到：时节在换
- 内部含义：节气、时令、阶段转换。
- 人生可能性：你处在一个转换点，旧节奏和新节奏正在交替。
- 命理触发：接近节气边界，或今日节气对五行权重影响明显。
- 适合：复盘上一段，调整下一段节奏。
- 不宜：用旧节奏处理新阶段。
- 短文案：时节在换，节奏也要换。
- 展开文案：节气节点代表外部时间在转。很多不适不是方向错，而是节奏没跟上。先调整节奏，再判断路况。

### 31. `conflict_marker`

- 用户看到：两股力相冲
- 内部含义：冲突、冲煞、摩擦。
- 人生可能性：你和某人、某事、某个环境之间存在拉扯。
- 命理触发：今日冲煞、risk 上升、关系或方向问题中出现冲突信号。
- 适合：避开正面硬碰，先拆开议题。
- 不宜：在情绪最强时做最终判断。
- 短文案：有冲，不宜硬碰。
- 展开文案：冲不是坏命，而是两股力方向不一致。先别正面撞，换一个说法、换一个时间，问题会更容易拆开。

### 32. `good_timing`

- 用户看到：顺势通行
- 内部含义：宜、顺势、时机较合。
- 人生可能性：今天有一段相对顺的路，可以做轻量推进。
- 命理触发：宜项命中当前意图，risk 低于阈值，support 或 momentum 不弱。
- 适合：推进一件能收口、能反馈的事。
- 不宜：把顺势当成保证。
- 短文案：顺势可行，先走一格。
- 展开文案：顺势通行代表今天有一小段路比较亮。它不是保证成功，而是提醒你可以借这个时机做一件具体、轻量、可回收的事。

## 4. 接入建议

1. `asset_manifest.json` 保持资源级字段，不直接塞长文案。
2. 新增前端或后端内容表：`icon_id -> visible_label -> short_copy -> expanded_copy -> do -> avoid -> trigger_hint`。
3. 规则引擎只决定出现哪些 icon、落在哪个人生阶段节点、推荐哪条路线。
4. 大模型只在固定字段内润色，不允许新增路线、不允许改变 icon 结论。
5. 每张地图默认展示 10-13 个 icon；摘要只显示当前最关键 1 个选择，其他作为“另一种可能”。
