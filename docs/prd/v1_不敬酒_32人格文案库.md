## 1. 语言标准

### 1.1 概述版

用于卡片第一屏。

字段：
- 人格名。
- 编码和四轴。
- 一句话。
- 表面状态。
- 内核状态。
- 酒桌表现。
- 高危瞬间。

句式：

```text
看起来在 A，其实在 B。
这类人不是 C，而是 D。
他们最擅长 E，但最容易在 F 失守。
```

### 1.2 详细版

用于下拉详情。

字段：
- 谁是这个人格。
- 他们如何不敬酒。
- 优势。
- 盲点。
- 推荐拒酒方式。
- 命理映射。

句式：

```text
拥有这种不敬酒人格的人，通常会先处理局面，再处理酒杯。
他们并不一定讨厌社交，但他们讨厌边界被别人临时改写。
当压力上来时，他们的第一反应不是喝，而是重新定义这一轮互动。
```

### 2.1 HYJK 男｜小圆审酒官水母

四轴：
- 硬挡。
- 预防。
- 借力。
- 控场。

概述版：
- 一句话：看起来在敬酒，其实在做酒精合规审查。
- 表面状态：他坐得很稳，话也不多。
- 内核状态：每一轮劝酒都要先过规则。
- 酒桌表现：先讲清楚自己喝什么、喝多少、怎么敬。
- 高危瞬间：对方用身份压人，说这杯必须给面子。

详细版：

拥有这种人格的人，不是不懂礼数，而是不接受别人临时修改他的边界。他在酒桌上的第一反应，是先把规则讲明白，再给对方面子。

他的优势是稳定、清楚、有组织感。只要他先开口，这桌酒就很难进入失控状态。问题是，他的冷静容易被看成扫兴，尤其遇到喜欢热闹的人，会被误解成不合群。

推荐方式：开局就声明边界，把酒换成茶或饮料，并给出一个正式祝词。不要等到第三轮再解释。

命理映射：日主偏强，食伤可用，比劫或印星能借力，日主不弱，形成 `H + Y + J + K`。
