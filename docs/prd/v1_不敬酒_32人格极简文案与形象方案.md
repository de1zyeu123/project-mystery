## 1. 页面文案层级

第一层结果页只展示：
- 人格名。
- 一句话。
- 核心态度。
- 酒桌表现。
- 高危瞬间。

下拉详情展示：
- 完整解读。
- 为什么是这个人格。
- 拒酒建议。
- 命理气质摘要。

不展示：
- `HYJK` 等内部编码。
- 四轴字母。
- 复杂八字排盘。
- “NPI / MBTI”字样。


```text
你并不是不会圆场，只是不想把时间浪费在反复解释上。
一旦对方越线，你会用最短的话把这一轮停掉。
```

```text
连续三次被点名起哄。
```

下拉详情：

```text
这种人格的边界很硬。前两次他可能还能客气，第三次开始，酒桌气氛会被他直接切断。

优势是压得住场，别人不容易继续进攻。风险是强度太高，容易把普通劝酒变成公开冲突。

拒酒建议：把硬话提前软化。先说清楚「我今天不喝」，再补一句祝词，不要等怒气满格才开口。
```

```text
日主偏强，表达星明显，边界感强，外部缓冲少。
```
