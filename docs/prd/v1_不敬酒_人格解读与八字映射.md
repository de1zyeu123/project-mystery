# 不敬酒人格解读与八字映射

日期：2026-05-31

状态：工作版，供逐图审核。

用途：
- 给 32 张人格卡配人格描述。
- 定义生辰八字到 16 型编码的映射逻辑。
- 定义性别如何扩展为 32 张卡。

说明：
- A/B/C 是视觉风格，不改变人格逻辑。
- 32 张主卡先以 B 套「胖圆贴纸派」命名为准。
- A 套和 C 套沿用同一人格解读，只替换视觉名字和画风。

## 1. 人格描述格式

参考 MBTI 人格卡的表达方式：
- 一个名称。
- 一个强反差短句。
- 一个可截图传播的判断。

本产品固定使用四段：

| 字段 | 用途 |
|---|---|
| 卡名 | 分享卡标题 |
| 反差短句 | 类似 MBTI meme 里的短评 |
| 人格解读 | 解释这个人为什么这样不敬酒 |
| 高危瞬间 | 最容易被劝酒突破的场景 |

文案公式：

```text
看起来是 A，其实是 B。
表面在 C，内心已经 D。
不是不懂礼貌，是不把边界外包。
```

## 3. 生辰八字到人格编码

### 3.1 输入

用户输入：
- 出生日期。
- 出生时间。
- 出生地。
- 性别。

系统计算：
- 年柱。
- 月柱。
- 日柱。
- 时柱。
- 日主。
- 十神分布。
- 五行分布。
- 日主强弱。
- 是否缺时辰。

### 3.2 输出

输出由两步决定：

```text
八字特征 -> 四轴编码 -> 16 型人格
性别输入 -> 男 / 女视觉变体
```

最终结果：

```text
人格编码 + 性别 = 32 张卡之一
```

示例：

```text
HYJK + 男 = 小圆审酒官水母
HYJK + 女 = 小圆冷章官水母
```

## 4. 四轴映射规则

### 4.1 H / S：冲突姿态

判断用户在劝酒压力下是硬挡还是软挡。

| 结果 | 命理信号 | 酒桌表现 |
|---|---|---|
| H 硬 | 日主偏强；比劫旺；金火偏旺；食伤能出 | 直接设边界，拒绝明确 |
| S 软 | 日主偏弱；印财偏重；水土偏旺；官杀压身 | 先圆场、给台阶、降低冲突 |

### 4.2 Y / A：反应时机

判断用户是开局预防，还是熬到后面处理。

| 结果 | 命理信号 | 酒桌表现 |
|---|---|---|
| Y 预 | 食伤透出；日主有根；火木偏旺；时柱主动 | 开局换杯、定规则、先发制人 |
| A 熬 | 官杀压力强；印星重；土水偏旺；日主被耗 | 先忍、先观察、被逼急再处理 |

### 4.3 J / D：社交资源

判断用户是借外部资源挡酒，还是自己处理。

| 结果 | 命理信号 | 酒桌表现 |
|---|---|---|
| J 借 | 比劫旺；印星可用；月柱支持强；合多于冲 | 借同桌、规则、健康理由、群体氛围 |
| D 单 | 比劫弱；冲多于合；日支孤立；外部支持弱 | 自己扛、自己拒、自己离场 |

### 4.4 K / P：结局动作

判断用户最后是控场，还是撤离。

| 结果 | 命理信号 | 酒桌表现 |
|---|---|---|
| K 控 | 食伤可用；印星能托底；日主不弱；金木结构清楚 | 转话题、改规则、把局面控回来 |
| P 跑 | 财官压力重；日主偏弱；水旺或驿马动；冲动位强 | 找电话、洗手间、提前撤、下线 |

## 5. 打分方法

每个轴独立打分。

基础规则：
- 强信号：+2。
- 弱信号：+1。
- 对侧强信号：-2。
- 对侧弱信号：-1。

决策：
- 分数 > 0，取左侧编码。
- 分数 < 0，取右侧编码。
- 分数 = 0，用 tie-breaker。

Tie-breaker：

| 轴 | 平分时规则 |
|---|---|
| H / S | 阳日干取 H，阴日干取 S |
| Y / A | 时柱可用且为寅卯巳午申酉取 Y，否则取 A |
| J / D | 比劫 + 印星数量大于财官食伤取 J，否则取 D |
| K / P | 日主不弱取 K，日主偏弱取 P |

缺出生时间：
- 不使用时柱。
- 只输出基础精度。
- Y/A 和 K/P 的置信度降低。
- 结果页显示「精度：基础」。

## 7. 性别到 32 张卡

性别不改变编码。

| 输入性别 | 处理方式 |
|---|---|
| 男 | 使用男版卡名、男版头像、男版文案口吻 |
| 女 | 使用女版卡名、女版头像、女版文案口吻 |
| 未提供 | 默认中性版，或要求用户补充 |

原则：
- 性别只影响视觉皮肤。
- 性别不影响八字轴分数。
- 同一编码的男女版是同一人格的两张卡。

## 9. 产品边界

必须显示：
- 这是娱乐型人格结果。
- 不判断真实酒量。
- 不鼓励饮酒。
- 不提供医学建议。

不能输出：
- 你命里必须喝。
- 你天生能喝。
- 不喝会倒霉。
- 喝酒能转运。
