## 1. 卡片目标

卡片只承担第一层传播。

目标：
- 一眼看懂人格。
- 一眼看懂拒酒方式。
- 一眼能截图分享。
- 不在第一屏解释完整八字逻辑。

不做：
- 长篇命理解释。
- 完整八字排盘。
- 酒量判断。
- 地图。

## 2. 版式结构

自上而下：

1. 产品名和风格线。
2. 人格名。
3. 编码和四轴标签。
4. 三个性格标签。
5. 人格 icon。
6. 身份信息。
7. 一句话短评。
8. 人格概述。
9. 高危瞬间。
10. 下拉展开入口。

## 3. 文案层级

### 3.1 标题

用途：
- 让用户记住这张卡。
- 像 MBTI 的人格名。

男版：

```text
棱面审酒官水母
```

女版：

```text
冷棱纪检官水母
```

### 3.2 编码行

用途：
- 给人格测试感。
- 对应四轴逻辑。

```text
HYJK｜硬挡 · 预防 · 借力 · 控场
```

### 3.3 标签

用途：
- 快速抓性格。
- 三个标签足够。

男版：

```text
酒桌纪检委 / 规则先行 / 不喝但有礼
```

女版：

```text
冷脸盖章 / 边界稳定 / 礼貌拒收
```

### 3.4 一句话

用途：
- 参考 MBTI meme 句式。
- 用反差表达，不做解释。

男版：

```text
看起来在敬酒，其实在做酒精合规审查。
```

女版：

```text
脸上没表情，桌上全是她的红章。
```

### 3.5 人格概述

用途：
- 第一层解释。
- 控制在 2 句内。

男版：

```text
他不是扫兴，是先把规则摆上桌。
谁劝、劝几轮、喝什么，都要重新定义。
```

女版：

```text
她不靠吵赢，而是靠冷静冻结局面。
酒杯还没递到手里，边界已经盖章。
```

### 3.6 高危瞬间

用途：
- 给用户代入感。
- 也是下拉详情入口。

男版：

```text
对方用身份压人，说这杯必须给面子。
```

女版：

```text
长辈或领导用玩笑包装命令。
```

### 3.7 下拉入口

用途：
- 提示还有第二层。
- 第一层不塞满。

```text
下拉展开：为什么是这个人格 / 如何拒酒 / 饭局风险点
```

## 4. 第二层内容

- 为什么是这个人格。
- 八字信号摘要。
- 不敬酒策略。
- 可复制话术。
- 饭局风险点。
