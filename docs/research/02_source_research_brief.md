# Source Research Brief

## Research Question

What source base can support a professional daily 玄学 dialogue app?

The source base must support:
- Correct calendar conversion.
- Correct 干支 / 节气 / 五行 logic.
- Traceable daily interpretations.
- Explainable personalization.
- Safe emotional support.

## Acceptance Criteria

A source can enter MVP only if it passes:
- Function fit.
- Active enough maintenance.
- License fit.
- Testability.
- Explainability.
- Peter approval.

An answer can enter the app only if it has:
- Calculation trace.
- Rule source.
- Plain-language interpretation.
- Supportive tone.
- No deterministic harm claim.
