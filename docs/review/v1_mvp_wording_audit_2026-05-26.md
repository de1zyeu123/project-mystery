# 人生路况 MVP 文案审核记录

日期：2026-05-26

## 审核角色

- Peter：命理大师视角，检查是否符合命理原则，不做绝对吉凶判断。
- 资深算命用户：C 端用户视角，检查是否好懂、有趣、可行动。
- Bill：产品经理视角，检查是否短、清楚、可复用。

## 文案原则

1. 不说“必然发生”，只说“今天更适合怎样走”。
2. 不用恐吓式吉凶词，改用路况词：绿灯、堵点、绕行、变道、降速。
3. 每句话服务行动，不做长解释。
4. 保留命理依据：干支、日主五行、节气、宜忌、冲煞、人生事件。
5. 大模型只改短文案，不改路线、分数、地图和判断结果。

## 已调整范围

- 输入页：改为“生成今日导航”，降低工具感。
- 结果页标题：改为短句池，如“路开了，稳走”“先稳住，别抢”。
- 三条路线：每类问题都有 A/B/C 三路短文案池。
- 快速路高风险状态：新增“欲速不达”“前方压车”“别抢红灯”等变体。
- 行动卡：将“收益/代价”改为“拿到/消耗”。
- 指标词：将“动力、稳定、助力、阻力”改为“行动火候、底盘稳度、顺风助力、堵点指数”。
- 分享按钮：改为“生成分享卡”。

## 结果可能性

当前每类问题都有：

- A 稳妥路：5 种短文案
- B 快速路：5 种短文案
- C 绕行路：5 种短文案

因此单一问题类型的可见短文案组合为：

```text
5 x 5 x 5 = 125 种
```

已覆盖问题类型：

- 今日综合
- 事业推进
- 关系沟通
- 财富承诺
- 情绪支持
- 方向选择

## 审核结论

当前文案达到 v1 MVP 标准：俏皮、克制、可行动，不直接断命。下一步应继续扩大命理规则库，而不是先扩大随机文案。
