# Product Spec Handoff

Status: template. Bill fills this after Peter and Andrew complete source review.

## Product Goal

Build a daily 玄学 dialogue app used by 1M users daily.

## Target User

User needs:
- Daily emotional support.
- Practical guidance.
- Professional 玄学 framing.
- Personal relevance.
- Low-friction conversation.

## Core Use Case

User opens the app daily and asks:
- 今天运势如何？
- 今天适合做什么？
- 我现在很焦虑，怎么看？
- 今天感情 / 工作 / 财务要注意什么？

## MVP Scope

Included:
- Daily 运势 card.
- Personal profile: birth date, birth time, gender, location/timezone.
- Calendar/BaZi calculation.
- Five-element interpretation.
- Dialogue follow-up.
- Traceable reasoning panel for internal review.

Excluded for MVP:
- Medical advice.
- Financial trading advice.
- Deterministic life claims.
- Relationship manipulation.
- High-stakes decisions.
- Paid ritual/法事 claims.

## Experience Principles

Tone:
- Friendly.
- Calm.
- Professional.
- Direct.
- Non-fear-based.

Answer structure:
1. 今日结论.
2. 关键原因.
3. 行动建议.
4. 情绪支持.
5. 可追问方向.
